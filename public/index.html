GWI Reconnaisance Toolkit

Welcome fellow pen-testers, GWI provdies a suite of reconnaisance tools accessable through a graphical user interface.
This isn't just a GUI front to your favorite recon tools, but additionally allows you to manage scan execution and to store scan reports.

The activities we currently support include:

  • Port Scanning
    - This uses nmap port scanning software to assist in fingerprinting networks
  • DNS Reconnaisance
    - From DNS lookups to DNS Zone transfers, this gathers as much data from DNS records
  • Firewall/IDS Scanning
    - This uses fragroute or wafw00f to fingerprint Firewalls and Network-based Intrusion Detection Systems

Integrated into our suite, we provide:

  • Concurrent Scan Execution
    - Users can execute multiple scans at the same time
  • Report Management
    - Users have access to a responsive report system where you can view, maximise, delete and order scan reports
  • Persistent Storage
    - We store and manage scan results so you don't have to

For developers who prefer the CLI, we provide a REST API:

  • Scan execution
    - we provide a simple method to submit scans for the GWI Suite to execute
  • Scan status
    - developers can query GWI for the status of any submitted query
  • Scan reports
    - provides a simple structured JSON formatted file containing scan reports
